# Projeto Cadastro de Clientes usando VueJS 3 com Vite

## Necessário versão 18 do node para rodar o projeto

## Para instalar o projeto
```
npm install
```

## Desenvolvimento
```
npm run dev
```

## Build para produção
```
npm run build
```

## Ambientes

### Local
O ambiente local vai rodar em:
http://localhost:8080
